# node-mongodb
node+mongodb 建站demo

开发

$ git clone git@github.com:seric705/node-mongodb.git

$ cd node-mongodb

$ npm install

$ bower install

$ node app.js


内容列表页:http://localhost:3000/admin/list

内容详情页:http://localhost:3000/movie/+文章id

内容录入页:http://localhost:3000/admin/movie

首页:http://localhost:3000
